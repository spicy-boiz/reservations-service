## Usage
From within the root directory:
Run the following commands then open localhost at port 3000
```sh
npm install
npm run seed
npm run server
npm run build
```

## Requirements

An `nvmrc` file is included if using [nvm](https://github.com/creationix/nvm).

- Node 6.13.0
